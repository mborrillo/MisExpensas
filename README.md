# Mis Expensas
---
## Descripción:

Herramienta de análisis automático de liquidaciones de expensas en formato PDF.
Convierte una liquidación compleja en un dashboard claro con indicadores financieros, estructura de gastos y hallazgos relevantes.

## Características
- ✔ procesamiento local (no sube datos a servidores)
- ✔ dashboard interactivo
- ✔ indicadores clave
- ✔ detección de inconsistencias
- ✔ insights automáticos
- ✔ exportación de resultados

## Versiones
- ✔ Basic
- ✔ Pro
- ✔ Premium

## Instrucciones
- Descargar HTML (desde carpeta recursos)
- Abrir en Navegador
- Cargar PDF de Muestra (desde carpeta recursos)
- Dashboard listo para visualizar

## Aclaraciones importantes
El archivo se ejecuta en local, sin necesidad de estar conectado a la red. Los datos, no se envian a ninguna plataforma ni servidor.
